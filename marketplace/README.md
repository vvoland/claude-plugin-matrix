# vvoland Marketplace

This directory contains the contents for a **separate** Claude Code plugin marketplace repository.

It intentionally contains only the marketplace catalog (no plugin code). The Matrix plugin itself lives in:

- https://github.com/vvoland/claude-plugin-matrix

## Install

From a Claude Code session:

```text
/plugin marketplace add vvoland/<marketplace-repo>
/plugin install matrix@vvoland
```
